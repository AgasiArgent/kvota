---

Интеграция с SmartLead для отслеживания эффективности email-рассылок. Доступно для владельцев организации.

## Что нового

- **Синхронизация с SmartLead** — автоматический импорт данных о кампаниях
- **Группировка по компаниям** — кампании объединяются по префиксу
- **Ключевые метрики:**
  - Лиды — общее количество контактов
  - Ответов — количество ответов
  - Теплые — позитивные ответы
  - Заявки — назначенные встречи
- **Конверсии (CR)** — автоматический расчёт воронки
- **Раскрывающиеся группы** — клик по компании показывает детали кампаний

## Как использовать

1. Откройте «Email кампании» в боковом меню (раздел «Маркетинг»)
2. Нажмите «Sync» для загрузки данных из SmartLead
3. Кликните на строку компании чтобы раскрыть детали
4. Используйте ИТОГО для просмотра общих показателей
