Внедрена новая система идентификаторов для коммерческих предложений и позиций.

## Новый формат номеров

**Формат КП:** `MBR-1734170432987-001`

- `MBR` - код компании-продавца (3 буквы)
- `1734170432987` - временная метка создания
- `001` - порядковый номер КП

**Формат позиции:** `MBR-1734170432987-001-01`

- Номер КП + порядковый номер позиции в таблице

## Коды компаний-продавцов

| Компания          | Код |
| ----------------- | --- |
| МАСТЕР БЭРИНГ ООО | MBR |
| РадРесур ООО      | RAR |
| ЦМТО1 ООО         | CMT |
| GESTUS            | GES |
| TEXCEL            | TEX |

## Преимущества

- Уникальные идентификаторы для каждого КП
- Автоматическая нумерация позиций
- Легко отслеживать историю изменений
- Удобный поиск по номеру
