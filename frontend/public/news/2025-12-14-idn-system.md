Внедрена новая система идентификаторов для коммерческих предложений и позиций.

## Новый формат номеров

**Формат КП:** `MBR-1234567890-2025-1`

- `MBR` - код компании-продавца (3 буквы)
- `1234567890` - ИНН клиента
- `2025` - год создания
- `1` - порядковый номер КП за этот год

**Формат позиции:** `MBR-1234567890-2025-1-01`

- Номер КП + порядковый номер позиции в таблице

## Коды компаний-продавцов

| Компания          | Код |
| ----------------- | --- |
| МАСТЕР БЭРИНГ ООО | MBR |
| РадРесур ООО      | RAR |
| ЦМТО1 ООО         | CMT |
| GESTUS            | GES |
| TEXCEL            | TEX |

## Преимущества

- По номеру КП сразу видно клиента (ИНН) и год
- Автоматическая нумерация для каждого года
- Уникальные идентификаторы для всех позиций
- Удобный поиск и отслеживание
