Внедрена новая система идентификаторов для коммерческих предложений и позиций.

## Новый формат номеров

**Формат КП:** `МБС-1234567890-001`

- `МБС` - код поставщика (3 буквы)
- `1234567890` - ИНН клиента
- `001` - порядковый номер для этого клиента

**Формат позиции:** `МБС-1234567890-001-01`

- Номер КП + порядковый номер позиции

## Преимущества

- Легко идентифицировать клиента по номеру КП
- Автоматическая нумерация для каждого клиента
- Уникальные идентификаторы для всех позиций
- Удобный поиск и отслеживание
