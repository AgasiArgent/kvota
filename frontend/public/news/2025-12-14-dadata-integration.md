Добавлена интеграция с сервисом DaData для автоматического заполнения данных клиента по ИНН.

## Что нового

При создании нового клиента теперь достаточно ввести ИНН - система автоматически подтянет:

- Полное наименование компании
- КПП и ОГРН
- Юридический адрес с индексом
- ФИО и должность директора

## Как использовать

1. Перейдите на страницу создания клиента
2. Введите ИНН в поле "Быстрый поиск по ИНН" вверху страницы
3. Нажмите "Найти" или Enter
4. Данные заполнятся автоматически

Работает для организаций (10 цифр) и ИП (12 цифр).
