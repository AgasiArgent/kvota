title: 'Конструктор списка КП'
date: '2025-12-22'
---

Новый мощный инструмент для работы со списком коммерческих предложений с настраиваемыми колонками и готовыми пресетами для каждого отдела.

## Что нового

- **65 колонок** — все данные из КП в одном месте
- **5 системных пресетов** — готовые настройки для отделов:
  - Продажи (ЕРПС)
  - Логистика
  - Бухгалтерия (КА)
  - Руководство (КП)
  - Полный вид
- **Пользовательские пресеты** — сохраняйте свои настройки колонок
- **Фильтрация и сортировка** — быстрый поиск нужных КП
- **Пагинация** — работа с большими объёмами данных

## Как использовать

1. Откройте страницу «Коммерческие предложения»
2. Выберите пресет из выпадающего списка или настройте колонки вручную
3. Нажмите «Сохранить пресет» чтобы сохранить свои настройки
4. Используйте фильтры в шапке таблицы для поиска
